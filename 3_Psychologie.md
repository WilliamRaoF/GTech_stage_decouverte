# 3. La psychologie du joueur

## Objectifs pédagogiques

À la fin de ce chapitre, vous serez capable de :

- Comprendre ce qui motive un joueur à jouer.
- Distinguer les motivations intrinsèques des motivations extrinsèques.
- Utiliser la théorie du Flow pour concevoir une difficulté adaptée.
- Identifier les différents profils de joueurs.
- Comprendre les biais cognitifs utilisés dans les jeux.
- Concevoir une expérience qui maintient l'engagement sur le long terme.

---

# 3.1 Pourquoi joue-t-on ?

Une erreur fréquente consiste à penser que les joueurs jouent uniquement pour gagner.

En réalité, les motivations sont nombreuses.

Un joueur peut jouer pour :

- se divertir ;
- apprendre ;
- explorer ;
- socialiser ;
- raconter une histoire ;
- relever un défi ;
- se détendre ;
- s'exprimer ;
- collectionner ;
- créer.

Deux joueurs peuvent jouer au même jeu pour des raisons totalement différentes.

Le rôle du game designer consiste à identifier ces motivations et à concevoir un système qui les satisfait.

---

# 3.2 Motivation intrinsèque et motivation extrinsèque

La psychologie distingue deux grandes familles de motivation.

## Motivation intrinsèque

La motivation intrinsèque vient de l'activité elle-même.

Le joueur agit parce qu'il trouve cette activité agréable.

Exemples :

- résoudre une énigme ;
- maîtriser un combo difficile ;
- construire une ville ;
- explorer une montagne ;
- apprendre un nouveau système.

La récompense est le plaisir de réaliser l'action.

---

## Motivation extrinsèque

La motivation extrinsèque repose sur une récompense extérieure.

Le joueur agit pour obtenir quelque chose.

Exemples :

- gagner de l'expérience ;
- obtenir une arme légendaire ;
- débloquer un succès ;
- monter dans un classement ;
- recevoir une monnaie virtuelle.

---

## Comparaison

Motivation intrinsèque :

> « Je joue parce que c'est amusant. »

Motivation extrinsèque :

> « Je joue pour obtenir une récompense. »

Les meilleurs jeux combinent généralement les deux.

---

# 3.3 La théorie de l'autodétermination

Les psychologues Edward Deci et Richard Ryan ont proposé que trois besoins fondamentaux favorisent une motivation durable.

## 1. L'autonomie

Le joueur doit avoir l'impression de faire ses propres choix.

Exemples :

- plusieurs chemins possibles ;
- différentes stratégies ;
- personnalisation du personnage ;
- liberté d'exploration.

À éviter :

- guider constamment le joueur ;
- imposer une seule solution.

---

## 2. La compétence

Le joueur veut progresser.

Chaque défi réussi renforce son sentiment de maîtrise.

Exemples :

- nouveaux pouvoirs ;
- amélioration de la précision ;
- ennemis plus complexes ;
- techniques avancées.

Un joueur qui ne progresse jamais abandonne rapidement.

---

## 3. L'appartenance

Le joueur apprécie les relations avec les autres.

Cela peut passer par :

- le multijoueur ;
- les guildes ;
- les PNJ attachants ;
- la coopération ;
- les communautés.

Même un jeu solo peut répondre à ce besoin grâce à des personnages crédibles.

---

# 3.4 La théorie du Flow

Le psychologue Mihály Csíkszentmihályi décrit le **Flow** comme un état mental où une personne est totalement absorbée par une activité.

Dans cet état :

- le temps semble passer plus vite ;
- la concentration est maximale ;
- la frustration est faible ;
- le plaisir est élevé.

Le Flow apparaît lorsque la difficulté est adaptée au niveau du joueur.

---

## Trop facile

Si le défi est inférieur aux compétences :

Le joueur s'ennuie.

Exemple :

Un boss éliminé sans effort.

---

## Trop difficile

Si le défi dépasse largement les compétences :

Le joueur se décourage.

Exemple :

Un niveau impossible dès le début.

---

## Zone de Flow

Le défi augmente progressivement en même temps que les compétences.

Le joueur apprend constamment sans être dépassé.

C'est cette progression qui maintient l'engagement.

---

# 3.5 Les courbes de difficulté

La difficulté ne doit pas être constante.

Elle évolue selon plusieurs principes.

## Montée progressive

Le joueur apprend une mécanique.

↓

Il la maîtrise.

↓

Le jeu introduit une nouvelle mécanique.

↓

Le joueur combine les deux.

C'est le modèle utilisé par de nombreux jeux de plateforme.

---

## Pics de difficulté

Certains moments sont volontairement plus difficiles.

Exemples :

- boss ;
- défis spéciaux ;
- énigmes complexes.

Ces pics sont généralement suivis d'une phase de récupération.

---

## Respiration

Le joueur a besoin de moments calmes.

Exemples :

- exploration ;
- dialogue ;
- boutique ;
- cinématique.

Sans respiration, la fatigue s'installe.

---

# 3.6 Les profils de joueurs (Bartle)

Richard Bartle a proposé une classification célèbre des joueurs, principalement appliquée aux jeux en ligne.

## Les Achievers (Performeurs)

Ils aiment :

- réussir ;
- terminer le jeu à 100 % ;
- obtenir les meilleurs équipements ;
- compléter les succès.

Ils apprécient les systèmes de progression.

---

## Les Explorers (Explorateurs)

Ils aiment :

- découvrir les secrets ;
- comprendre les systèmes ;
- visiter toutes les zones.

Ils apprécient les mondes ouverts et les environnements riches.

---

## Les Socializers (Sociaux)

Ils jouent pour interagir.

Ils aiment :

- discuter ;
- coopérer ;
- aider les autres ;
- créer une communauté.

---

## Les Killers (Compétiteurs)

Ils aiment :

- battre les autres joueurs ;
- dominer ;
- grimper dans les classements.

Ils apprécient les jeux compétitifs.

---

# 3.7 Les émotions recherchées

Tous les jeux ne cherchent pas à provoquer les mêmes émotions.

Quelques exemples :

## Tension

Jeux d'horreur.

Le joueur manque de ressources.

Le danger est permanent.

---

## Euphorie

Jeux d'action.

Le joueur enchaîne des actions spectaculaires.

---

## Curiosité

Jeux d'exploration.

Le joueur veut découvrir ce qui se cache derrière une porte ou une montagne.

---

## Satisfaction

Jeux de gestion.

Optimiser un système procure un fort sentiment d'accomplissement.

---

## Émerveillement

Jeux contemplatifs.

L'environnement devient une récompense en lui-même.

---

# 3.8 Les biais cognitifs utilisés en game design

Les jeux exploitent souvent certains mécanismes psychologiques.

Il est important de les connaître afin de les utiliser de manière éthique.

## L'effet de progression

Voir une barre d'expérience se remplir encourage à continuer.

Même une faible progression est motivante.

---

## L'aversion à la perte

Les êtres humains détestent perdre.

Perdre une série de victoires ou un objet rare est souvent plus marquant que gagner quelque chose d'équivalent.

Les jeux utilisent cette émotion avec prudence.

---

## La curiosité

Une récompense cachée pousse naturellement à explorer.

Exemples :

- coffre caché ;
- passage secret ;
- succès mystérieux.

---

## Les récompenses variables

Toutes les récompenses ne sont pas prévisibles.

Le hasard peut renforcer l'engagement.

Exemple :

Un ennemi rare laisse parfois tomber un objet exceptionnel.

Attention : un usage excessif peut créer des comportements problématiques. Le game designer doit privilégier une conception respectueuse du joueur.

---

# 3.9 Les récompenses

Une récompense n'est pas uniquement un objet.

On distingue plusieurs catégories.

## Récompenses mécaniques

- nouvelle arme ;
- compétence ;
- équipement.

---

## Récompenses narratives

- révélation ;
- cinématique ;
- nouveau personnage.

---

## Récompenses émotionnelles

- satisfaction ;
- surprise ;
- humour.

---

## Récompenses sociales

- titre ;
- classement ;
- reconnaissance des autres joueurs.

---

# 3.10 Les frustrations positives

Toutes les frustrations ne sont pas mauvaises.

Une frustration contrôlée peut renforcer le plaisir.

Exemple :

Le joueur échoue contre un boss.

Il comprend pourquoi.

Il adapte sa stratégie.

Il réussit ensuite.

La victoire est alors beaucoup plus satisfaisante.

À l'inverse, une frustration injuste (caméra défaillante, commandes imprécises, règles incohérentes) génère de l'abandon.

---

# 3.11 Étude de cas : Celeste

Pourquoi **Celeste** est-il souvent cité comme exemple de bon game design ?

- Les contrôles sont extrêmement précis.
- Les échecs sont fréquents mais rapides.
- Le joueur réapparaît immédiatement.
- Chaque niveau introduit une idée nouvelle.
- Les défis augmentent progressivement.

Le jeu est difficile, mais il donne toujours au joueur les moyens de réussir.

La difficulté est perçue comme juste.

# Exercices

## Exercice 1

Choisissez un jeu que vous aimez.

Identifiez :

- les motivations intrinsèques ;
- les motivations extrinsèques.

Lesquelles sont les plus importantes ?

---

## Exercice 2

Analysez une séquence de jeu.

À quels moments le joueur est-il :

- stressé ;
- détendu ;
- récompensé ;
- frustré ;
- surpris ?

Expliquez pourquoi.

---

## Exercice 3

Concevez une boucle de progression.

Répondez aux questions suivantes :

- Qu'apprend le joueur ?
- Comment devient-il plus compétent ?
- Quand introduisez-vous une nouvelle difficulté ?
- Quelle récompense reçoit-il après chaque étape ?

---

# À retenir

- Les joueurs n'ont pas tous les mêmes motivations.
- La motivation intrinsèque est plus durable que la motivation extrinsèque.
- Le Flow apparaît lorsque la difficulté suit la progression du joueur.
- Une bonne courbe de difficulté alterne défis et moments de respiration.
- Les récompenses peuvent être mécaniques, narratives, émotionnelles ou sociales.
- Les biais cognitifs doivent être utilisés avec responsabilité.
- Un bon game designer conçoit des systèmes qui encouragent la curiosité, l'apprentissage et le plaisir plutôt que la simple accumulation de récompenses.
